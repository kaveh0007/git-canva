GitHub contribution graphs are public signals developers use to show consistency. Every tool that paints them is either a CLI script with no visual feedback, or supports backdating only with no forward commit option. Nobody has built a proper canvas experience where you visually design your art and choose whether to fake it or earn it. That's the gap I have experienced.

Through git-canva I intend to fix that and create a tool developers can use for fun. 

Getting started with this on Monday, 18th May 2026.
